# MINS
Links to repos and research for Mobile Inelastic Neutron Scattering System


## Repositories
### INS Analysis
- [INS-Analysis](https://github.com/JoseACortes/INS-Analysis): Python package for the analysis of inelastic neutron scattering data.
- [INS-Analysis-FE](https://github.com/JoseACortes/INS-Analysis-FE): Frontend web app for the INS-Analysis package.
- [MCTalTimeBlur](https://github.com/JoseACortes/MCTalTimeBlur)

### Soil Sample Simulation
- [SimpleSoilSampleSim](https://github.com/JoseACortes/SimpleSoilSampleSim): Simple simulation of soil samples of the core sampling method.
